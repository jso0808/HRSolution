월소득액 = 급여액-비과세액(식대 10만원)

1.복리후생비
식대, 교통비, 통신요금 등 사업주가 근로자에게 제공하는 혜택

국민연금 = 월소득액 x 4.5%
건강보험 = 월소득액 x 3.495%
장기요양보험 = 건강보험 x 12.27%
고용보험 = 월소득액 x 0.9%
산재보험 = 일반회사=0

소득세 계산식 추가
간이세액표 엑셀 파일 가져와서 계산하기
	- 엑셀 파일 반영 질문
		-> 전부 반영X 엇비슷하게 반영
200만원부터 50만원씩 600만까지 공제대상가족수2인기준. 그 이상은 기본급/10

주민세(지방소득세) = 소득세 x 10%
